\doxysubsection{Detailed Description}
Manages user accounts, including creation, authentication, and persistence. 

This class handles all operations related to user accounts, such as adding, modifying, deleting, and authenticating users. It interacts with an \doxylink{class_api_client}{Api\+Client} to load and save user data from/to a webservice and uses a \doxylink{class_parser}{Parser} for data serialization/deserialization.
