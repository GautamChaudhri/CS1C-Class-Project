\doxysubsection{Detailed Description}
Provides functionality for parsing JSON data to C++ objects and vice-\/versa. 

The \doxylink{class_parser}{Parser} class handles the serialization and deserialization of various data structures used in the application, such as Shapes, User\+Accounts, and Testimonials, converting them to and from JSON string representations. It includes methods for both forward parsing (JSON to object) and reverse parsing (object to JSON). This class does not store any data itself and its copy/move operations are disabled.
